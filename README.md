# Esp32-s3-jammer-for-if-you-only-have-1-nRF24L0-PA-LNA-
***FOR EDUCATIONAL PURPOSES*** code is kinda wack// but it works. 
This code is designed exclusively for the ESP32-S3 paired with a single nRF24L01 transmitter. While this setup will not disconnect already paired Bluetooth devices, it may increase the time it takes for them to initially establish a connection.

Currently, the code only targets 3 channels, meaning it is not 100% effective as a sweeping jammer/tester. However, adding more channels to the sequence would be redundant, as a single nRF24L01 chip cannot effectively cover a wider spectrum simultaneously anyway.
